## Структура проекта

Общая структура:

```
noties/
	|- app/
	|	# содержит все относящиеся к приложению файлы - скрипты, шаблоны, стили
	|	|- client/
	|	|	# содержит модули, необходимые только клиентской части
	|	|- common/
	|	|	# содержит модули, используемые как на сервере, так и на клиенте
	|	|- server/
	|	|	# содержит модули, используемые только на сервере
	|	|	|- endpoints/
	|	|	|	# финальные классы, обрабатывающие каждый запросы к своей endpoint
	|	|	|- rest/
	|	|	|	# абстрактные вспомогательные классы для построения api
	|	|	|- api.js
	|	|	|	# файл, комбинирующий все endpoints в единое апи
	|	|- pages/
	|	|	# структура страниц сайта. здесь размещаются шаблоны как статических, так и динамических страниц.
	|	|	# каждая страница должна быть указана в карте сайта.
	|	|	# желательно сохранять в этой папке ту же структуру, которая будет представлена на сайте
	|	|	# здесь же лежат файлы index.js и style.less, относящиеся к соответствующим страницам.
	|	|	# они автоматически компилируются, минифицируются и собираются webpackом.
	|	|	# наличие index.js необходимо для корректной работы сборщика.
	|	|	# файлы стиля опциональны и requireятся из index.js
	|	|- templates/
	|	|	# здесь находятся общие для всего сайта шаблоны (например, футер и хедер)
	|	|- index.js
	|	|	# главный файл серверного приложения
	|	|- config.js
	|	|	# конфигурация сервера
	|	|- sitemap.js
	|	|	# карта сайта
	|- docs/
	|	# документация
	|- public/
	|	# содержит все статические файлы сайта. содержимое папки полностью автогенерируемо
	|- build.sh
	|	# скрипт для запуска сборки
	|- run.sh
	|	# скрипт для запуска сервера
	|- aliases
	|- webpack.config.js
	|	# конфигурация сборщика фронтэнда (много чёрной магии вуду)
```

## Пример структуры страниц сайта

Пример структуры страниц сайта в `pages` и соответствующий результат сборки в `public`:

```
noties/app/pages/
	|- index.handlebars
	|- index.js
	|- style.less
	|- profile/
	| 	|- index.handlebars
	|	|- index.js
	|	|- style.less
	|- editor/
	| 	|- index.handlebars
	| 	|- component.handlebars
	|	|- index.js
	|	|- style-light.less
	|	|- style-dark.less

noties/public/
	|- index.html
	|- bundle.js
	|- style.css
	|- profile/
	|	|- index.html
	|	|- bundle.js
	|	|- style.html
	|- editor/
	|	|- index.html
	|	|- bundle.js
	|	|- style.html
```
